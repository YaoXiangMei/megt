# 集成项目的各种小工具，能快速配置规范项目


### git commit 规范工具
```shell
megt-commitizen
```
### git commit message检查工具
```shell
megt-commitlint
```
### 添加changelog.md工具
```shell
megt-changelog
```
### 添加eslint工具
```shell
megt-eslint
```
### 添加editorconfig工具
```shell
megt-editorconfig
```
### 添加gitignore工具
```shell
megt-gitignore
```
